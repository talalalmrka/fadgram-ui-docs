Badge

Basic Usage:

Badge

Colors

replace {color} with the color name.
<span class="badge badge-{color}">badge</span>

Outline badge

<span class="badge badge-outline badge-{color}">badge</span>

Badge pill

<span class="badge pill badge-{color}">badge</span>

Sizes

replace {size} with the size (xs, sm, lg, xl).
<span class="badge badge-primary {size}">badge</span>
