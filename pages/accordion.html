Accordion

Requirments

this accordion plugin requires Alpine.js for installation read official documentation here Alpine js installation.

Apply accordion to Alpine js

import Alpine from 'alpinejs';
import accordion from 'fadgram-ui/alpine/accordion';
Alpine.plugin(accordion);
